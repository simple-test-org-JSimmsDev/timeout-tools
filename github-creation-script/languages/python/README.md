# Hello World Project

A simple "Hello, World!" Python application.

## Requirements
- Python 3.11+
- (Optional) Docker

## Setup

1. run The application locally 
    ```bash
    python app/hello_world.py
    ```

2. Run Tests (Windows)
    ```bash
    python -m pytest
    ```

3. Build and run with Docker
    ```bash
    docker build -t hello-world-app .
    docker run hello-world-app
    ```
